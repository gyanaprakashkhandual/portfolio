# Mega Jewelers Testing

End-to-end testing infrastructure for MegaJewelers including UI automation, API testing, performance testing, and enterprise-grade QA strategy for a complete e-commerce platform.

## Overview

Mega Jewelers testing framework provides a comprehensive end-to-end testing infrastructure for a large-scale e-commerce platform designed for jewelry businesses. The framework integrates UI automation, API validation, performance testing, and manual functional testing to ensure the reliability and scalability of the application. Using Selenium automation with Cucumber BDD architecture and Page Object Model design, the testing framework ensures maintainable and reusable test automation. Performance testing with Artillery validates system behavior during peak traffic periods while API testing with PyTest guarantees backend data integrity and system stability.

## Key Features

- Comprehensive UI automation testing using Selenium
- Behavior-driven testing using Cucumber
- API testing framework using PyTest
- Performance testing using Artillery
- Page Object Model architecture for scalable automation
- Custom HTML/CSS/JavaScript reporting dashboards
- Test data management using Google Sheets and Zoho Sheets
- Enterprise-level test execution and defect tracking
