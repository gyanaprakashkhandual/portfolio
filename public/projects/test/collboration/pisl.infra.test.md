# PISL Infra

Comprehensive end-to-end testing infrastructure for the PISL web application including UI, API, performance, and security testing with advanced visualization and reporting.

## Overview

PISL Infra is a complete automation and testing infrastructure designed to validate the reliability, performance, and security of the PISL web application. The system integrates UI testing, API testing, load testing, and security testing into a unified framework. Using modern tools like K6 and Grafana, it enables real-time performance monitoring while generating detailed and visually rich reports through custom SVG visualization. The project ensures that backend services, APIs, and user interfaces operate efficiently under real-world conditions while maintaining security and scalability.

## Key Features

- End-to-end UI automation testing
- API automation testing with structured test suites
- Advanced performance testing using K6
- Security testing and vulnerability analysis
- Real-time monitoring dashboards using Grafana
- Custom SVG-based performance report generation
- Automated test execution pipelines
- Detailed analytics and performance insights

## What Makes It Stand Out

The most unique aspect of PISL Infra is its custom SVG-based performance reporting system that transforms complex performance metrics into visually rich and interactive reports. Combined with live Grafana dashboards and automated testing pipelines, the system provides a complete view of application health and performance.
