# Rr Crop

A task and resource management platform designed for project allocation, workforce management, and operational tracking with integrated access control and collaboration tools.

## Overview

Rr Crop is a full stack task and workforce management platform designed to handle complex project allocations and team coordination. The system enables administrators to create projects, assign tasks, allocate workers, and track project progress in real time. It provides structured management of staff, resources, and project timelines through an intuitive dashboard interface. The platform focuses on improving productivity by providing centralized control over project planning, task distribution, and workforce management while ensuring secure access and scalable performance.

## Key Features

- Comprehensive task management system
- Project allocation and tracking across teams
- Workman and staff assignment to specific tasks
- Role-based access control and authentication
- Real-time project progress monitoring
- Interactive dashboards with analytics
- Full CRUD operations for tasks, projects, and staff
- Collaboration features for teams and project managers

## Features

- Project Allocation and Management

- Workforce and Staff Assignment

## What Makes It Stand Out

Rr Crop stands out by combining project management with workforce allocation in a single platform. The system enables organizations to efficiently distribute work among team members, monitor task progress, and manage resources through centralized dashboards and structured project workflows.

## Tech Stack

**Frontend:** React, Next JS, Tailwind CSS

**Backend:** Node.js, Express, MongoDB, JWT

**Infrastructure:** Vercel, Render, Cloudinary
