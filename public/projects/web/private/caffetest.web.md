# Caffetest

A web application for automating all the manual effort in automation testing.

## Overview

CafeTest is a comprehensive bug tracking and test management platform designed to automate and streamline software testing workflows. The application integrates modern full stack technologies with AI capabilities to simplify how QA teams manage projects, test cases, bugs, and reports. By combining structured test management with intelligent automation, CafeTest eliminates repetitive manual effort and improves testing efficiency.

The platform includes enterprise-grade capabilities such as role-based access control, project organization, sprint planning, and flexible dashboards. It also integrates AI APIs to automatically generate test cases from Selenium automation scripts and identify failures as bugs. With features like real-time synchronization, collaborative messaging, documentation support, and detailed reporting dashboards, CafeTest acts as a centralized platform for developers, testers, and project managers.

## Key Features

- AI Test Case Generation from Selenium Scripts
- Role-Based Access Management
- Multi-view Dashboard (Kanban, Split, Card, List)
- Real-time Auto-save with Keystroke Detection
- Built-in Messaging and Collaboration
- Theme Customization (Dark, Light, System)
- VS Code Extension Integration
- Import, Export and Sharing Features

## Features

## What Makes It Stand Out

The most unique feature of CafeTest is the VS Code extension integration that leverages Anthropic AI APIs to automatically generate test cases from Selenium automation scripts. It also detects failures during execution and categorizes them as bugs, significantly reducing manual effort in test creation and bug reporting. This makes CafeTest far more intelligent than traditional tools such as Jira and Zephyr.
